# Iris-Flower-Classification
Machine Learning classification project using the Iris dataset and K-Nearest Neighbors (KNN).



## Overview

This project classifies Iris flowers into three species:

* Setosa
* Versicolor
* Virginica

using the K-Nearest Neighbors (KNN) Machine Learning algorithm.

## Technologies Used

* Python
* Scikit-Learn
* NumPy

## Features

* Dataset Loading
* Train-Test Split
* KNN Classification
* Accuracy Evaluation
* Flower Species Prediction

## Result

Model Accuracy: 100%

## Dataset

Built-in Iris Dataset from Scikit-Learn.
